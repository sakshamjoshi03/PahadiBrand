# PahadiBrand

An AI-powered branding assistant for rural food producers.

## One-Line Description
PahadiBrand helps food-processing businesses generate brand names, product descriptions, packaging content, competitor insights, social media posts, and brand stories using AI.

## Tech Stack
- React
- CSS
- Express.js
- MongoDB
- Gemini API
- JWT Authentication
 

How to Run Backend Locally:

1. cd backend
2. npm install
3. Create .env
4. Add PORT=5000
5. npm run dev
